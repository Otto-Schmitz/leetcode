## English

Repository of **Java** solutions for [LeetCode](https://leetcode.com/) problems.

### Structure

- **Solution files**: `ProblemName_Number.java` (e.g. `AddTwoNumbers_2.java`)
- **`complement/`**: Helper classes used across problems (e.g. `ListNode` for linked lists)

### Build and run

Compile from the project root (so the `complement` package is found):

```bash
javac complement/*.java *.java
```

To run a class that has a `main` method:

```bash
java ClassName
```

### Requirements

- Java (JDK 8 or higher)
